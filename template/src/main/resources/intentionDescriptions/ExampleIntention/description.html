Example Intention Action

This intention action demonstrates how to create a custom intention that appears in the IDE's quick fix menu.

Usage

Place your cursor on a Java element and press Option+Enter (Mac) or Alt+Enter (Windows/Linux) to see available intentions.

Features

  • Appears in the intention menu for Java files
  • Provides context-aware suggestions
  • Can be customized based on the current element

Example

public class Example { public void method() { // Place cursor here and press Alt+Enter // to see the intention action } }

This intention action is part of the Example Plugin template and serves as a starting point for creating custom intentions.
